# ApexCRM Messaging Server

## Quick Railway Deployment

Real-time messaging server for ApexCRM using Socket.io

## Features
- ✅ Real-time messaging
- ✅ User presence tracking  
- ✅ Typing indicators
- ✅ Workspace isolation
- ✅ Video call invitations
- ✅ Auto-reconnection
- ✅ CORS configured for ApexCRM
